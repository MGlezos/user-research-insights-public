# ⚡︎⚡︎⚡︎ User Interview Insights ⚡︎⚡︎⚡︎

A free, open source tool for turning user interview recordings into clear, actionable insights in minutes.

Built for product designers, product managers, and researchers who want faster synthesis without heavy tools or subscriptions.

**Live demo:** [https://supersoniq-insights.vercel.app/](https://supersoniq-insights.vercel.app/)

---

## ⚡︎⚡︎⚡︎ What it does ⚡︎⚡︎⚡︎

Upload an audio or video recording from a user interview and get:

- Full transcript with speaker detection
- Automatic overview and summary
- Key insights and themes
- Positive and negative quotes
- Sentiment analysis over time and by speaker
- Raw transcript export as text

---

## ⚡︎⚡︎⚡︎ How it works ⚡︎⚡︎⚡︎

This app runs entirely on your own API keys and does not store uploaded files or transcript data.

### AssemblyAI (Voice to Text)

Used for transcription and analysis:
- Speaker labels
- Sentiment analysis
- High quality voice to text

Includes 330 free hours on the free tier.

### AI Summary and Insights

Used to generate summaries, insights, and quotes.

Supports:
- Claude
- Gemini
- OpenAI

About $5 in credits goes a long way.

---

## ⚡︎⚡︎⚡︎ How to use ⚡︎⚡︎⚡︎

1. Open the app [https://supersoniq-insights.vercel.app/](https://supersoniq-insights.vercel.app/)
2. Enter your API keys
   - AssemblyAI API key
   - AI provider API key of your choice
3. Upload an audio or video interview file
4. Click Transcribe and wait a few moments
5. Review the generated insights, quotes, sentiment, and full transcript
6. Copy or download the results as text

Each session is temporary. Nothing is saved.

## ⚡︎⚡︎⚡︎ What this is not ⚡︎⚡︎⚡︎

This is intentionally not a full SaaS product.

It does not include:
- User accounts
- Saved projects
- Team workspaces
- Dashboards
- Long term storage

Think of it as a strong starting point, not a finished platform.
